Первый пост

Блог собран на основе Jekyll. Вообще впечатлиния от использования Jekyll очень положительные. Тут вам и куча языков разметки, использование Liqiud, расширяемость, тэги, пейнджинг, sitemap, atom, а в итоге получаем простой набор html страничек которые можно хостить где угодно.

Этот блог к примеру висит на гитхабе. Гитхаб умеет сразу собирать ваш сайт на своей стороне из jekyll’а после каждого git push, только вот плагины он не подхватывает к сожалению. Поэтому репо исходников отдельно, рейк таск для деплоя и вперёд и с песнями.

Писать с нуля все было лень, за основу взял такую клевую штуку как Jekyll First Potion (там есть неплохой плагин для тэгов).
