Rails BDD Cheatsheet


Новое приложение 

  rails new bdd_app

Содержимое Gemfile 

  source 'http://rubygems.org'

  gem 'rails'

  group :production do
  end

  group :development do
    gem 'sqlite3'
    gem 'capistrano'
    gem 'ruby-debug19', :require => 'ruby-debug'
  end

  group :development, :test do
    gem 'rspec-rails'
    gem 'cucumber-rails'
  end

  group :test do
    #autotest stuff
    gem 'autotest-rails'
    gem 'autotest-growl'

    gem 'capybara'
    gem 'database_cleaner'
    gem 'gherkin', :git => 'https://github.com/aslakhellesoy/gherkin.git'
    gem 'launchy'
    gem 'email_spec'
    gem 'fabrication' #or factory_girl_rails
    gem 'timecop'
  end

Установка 

  bundle install
  rails g rspec:install
  rails g cucumber:install --rspec --capybara
  rails g email_spec:steps

Включение autotest/growl 

  echo "require 'autotest/growl'" >> .autotest

Игнорим лишнее 

  echo '.DS_Store' >> .gitignore #для Cappuccino

Настройка Email Spec 

  #env.rb for cucumber
  require 'email_spec/cucumber'
  #spec_helper.rb
  require "email_spec"

Настройка Capybara 

  #env.rb for Cucumber
  Capybara.register_driver :selenium do |app|
    Capybara::Driver::Selenium.new app, :browser => :chrome
  end

Запуск autotest 

  bundle exec autotest

Все, можно приступать к работе
